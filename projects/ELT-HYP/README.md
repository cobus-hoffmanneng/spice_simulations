# ELT-HYP — Eltwin E-CONTACTOR / soft-starter sub-circuit sims

Home for the SPICE decks that check specific claims from Nikola's review of the Eltwin
soft-starter board (project `01 - Projects/ELT-HYP/` in the vault).

Targeted **sub-circuit** sims only (not whole-board): e.g. the aux-supply divider,
opto-trigger LED current across the 20:1 input range, and SCR conduction-loss / thermal.
Nikola will populate this folder — hand-written netlists as `*.cir` (source), outputs ignored.

> Do NOT copy client originals (the Eltwin PDF, etc.) into this repo — this is for our own
> simulation sources, not client-supplied files.
